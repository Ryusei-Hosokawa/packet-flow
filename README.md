# PacketFlow

TCP/IP プロトコルをインタラクティブに学べる可視化ツール

## Demo

**[PacketFlow を試す](https://packet-flow.vercel.app/)**

## 概要

「マスタリングTCP/IP」で学んだ理論を、実際に手を動かして理解を深めるために開発した学習ツールです。アニメーションやインタラクティブな図解を通じて、ネットワークの基礎概念を視覚的に学べます。

## 機能

### OSI / TCP-IP モデル
OSI参照モデル（7層）とTCP/IPモデル（4層）の対応関係を図解。各層の役割とプロトコルを確認できます。

### 3ウェイハンドシェイク
TCP接続確立の流れをステップバイステップでアニメーション表示。SYN、SYN+ACK、ACKの各段階を視覚的に理解できます。

### パケット構造
TCP/UDP/IPヘッダの各フィールドをインタラクティブに探索。フィールドをクリックすると詳細な説明が表示されます。

### カプセル化
データが各層を通過する際のヘッダ追加（カプセル化）と除去（非カプセル化）の過程を可視化します。

### TCP vs UDP 比較
TCPとUDPの違いを比較表とユースケースで解説。信頼性と速度のトレードオフを理解できます。

### 用語ツールチップ
専門用語をクリックすると詳細な説明がポップアップ表示。約50の用語を収録しています。

## 技術スタック

- **Framework**: SvelteKit + Svelte 5 (Runes)
- **Styling**: Tailwind CSS v4
- **Language**: TypeScript
- **Hosting**: Vercel
